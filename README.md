# claw-portfolio

A static personal portfolio website for Abhigyan, built with React, TypeScript, and Vite.

## What it includes

- Hero section with a short professional intro
- Skills summary
- Education
- Work experience across Walmart, Rakuten, Podeum, and Morgan Stanley
- A note about testing Hermes and OpenClaw agents

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## GitHub Pages

This repo is intended to be hosted as a static GitHub Page at:

`https://abster12.github.io/claw-portfolio/`

For Pages, use the legacy direct serve/static branch style or GitHub Pages static hosting, not a deploy workflow.
